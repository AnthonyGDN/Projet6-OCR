Pour démarrer le projet :
1 - Ouvré le dossier "backend", démarré "Git Bash" dans ce dossier puis entré la commande "npm install" 
2 - Puis "npm install express mongoose bcrypt jsonwebtoken dotenv multer mongoose-unique-validator sharp cors" 
3 - Enfin "npm start"
4 - Ouvré le dossier "frontend", démarré "Git Bash" dans ce dossier puis entré la commande "npm install" 
5 - Puis "npm install express mongoose bcrypt jsonwebtoken dotenv multer mongoose-unique-validator sharp cors" 
6 - Enfin "npm install start"
7 - La page du site devrais s'ouvrir quelques seconde après
