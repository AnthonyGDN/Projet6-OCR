Pour démarrer le projet :
1 - Ouvré le dossier "backend", démarré "Git Bash" dans ce dossier puis entré la commande "npm install" 
2 - Maintenant faite "npm start"
3 - Ouvré le dossier "frontend", démarré "Git Bash" dans ce dossier puis entré la commande "npm install" 
4 - Maintenant faite "npm start"
5 - La page du site devrais s'ouvrir quelques seconde après
